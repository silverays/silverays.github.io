## 6. Projects & Case Studies

### 6.1 Bundle SKU Feature — Motorola Solutions
- **Role:** Senior Frontend Engineer
- **Year:** 2023
- **Duration:** 6 months
- **Tech:** React, Oracle Storefront Framework (OSF), Oracle Commerce Cloud (OCC), TypeScript
- **Summary:** Led Bundle SKU feature launch, improving B2B purchasing by 75%. Earned Sprint Champion award.
- **Problem:** Customers buying bundled products had to add each component individually, causing cart abandonment.
- **Key decision:** Frontend-only composition within OSF constraints — no backend changes permitted.
- **Challenge:** Synchronising state across multiple SKU configurations within a bundle. Performance issue with individual API calls resolved via batching.

### 6.2 Vue File Agent (Open Source)
- **Role:** Creator
- **Year:** 2019
- **Tech:** Vue.js, JavaScript, HTML5, CSS3

- **Summary:** Vue.js file upload component with drag-and-drop, multi-file support, image previews.
- **Architecture:** Plugin-based — core handled file logic, renderers handled UI. Event-driven state machine for file lifecycle.

## 11. Site Architecture (Current: Astro Rocket Theme)

The current website uses the **Astro Rocket** theme (v2.0.0) by Hans Martens.

### Tech Stack
- **Framework:** Astro 7
- **Styling:** Tailwind CSS 4 with OKLCH color tokens
- **Icons:** Lucide (UI) + Simple Icons (brand) via `astro-icon`
- **Content:** Astro Content Collections with Zod validation
- **Search:** Pagefind (post-build indexing)
- **Dark mode:** 3-state (system/light/dark), no flash
- **Themes:** 12 switchable colour themes (orange, amber, lime, emerald, teal, cyan, sky, blue, indigo, violet, purple, magenta)
- **Fonts:** Manrope (body), Outfit (display), JetBrains Mono (code) via Fontsource

## 13. Design Direction Notes

From the earlier design brief, the preferred aesthetic is **"Technical Memo"** — an editorial engineering portfolio with:
- Strong typography, generous spacing, disciplined grid
- Warm neutral backgrounds, near-black text, one restrained accent colour
- Small technical annotations and diagrams
- Subtle, purposeful animation
- Excellent mobile layouts
- Accessible colour contrast and keyboard navigation

**Avoid:** Neon cyberpunk, excessive tech logos, animated backgrounds, fake terminals, 3D effects, skill bars, generic AI claims, invented metrics.
