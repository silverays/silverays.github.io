## Visual concept

Blend:

- Editorial engineering portfolio
- Product studio
- Technical documentation clarity
- Subtle founder personality

Use:

- Strong typography
- Spacious layout
- Crisp cards and diagrams
- Monochrome or restrained base palette
- One distinctive accent colour
- Fine grid/diagram details
- Real UI screenshots
- Code fragments used selectively
- Clear portrait photography

Avoid:

- Humanoid robots
- AI brains
- Excessive circuit-board backgrounds
- Generic stock teams
- Fake office scenes
- Unrelated 3D blobs
- Random dashboard mockups that are not Safraz's work

## Image-generation prompt: abstract hero background

> Create a restrained editorial technology background for a senior software engineering portfolio. Use a clean architectural grid, subtle node connections, small code-like geometric marks and layered system diagrams. The mood should feel experienced, precise and product-focused—not cyberpunk. Leave generous negative space for headline text. No robots, no brains, no people, no readable text, no company logos, no neon overload.

## Image-generation prompt: product recovery concept

> Create a sophisticated isometric editorial illustration showing an unfinished software product becoming a reliable production system. On the left, fragmented interface panels, missing workflow connections and inconsistent components; in the centre, an engineering audit and structured milestone path; on the right, a coherent application with connected architecture, complete workflows and polished UI. Minimal, professional, technical, restrained colour palette, no readable text, no robots.

## Image-generation prompt: AI-assisted engineering process

> Create a clean horizontal systems illustration for a portfolio website showing six stages: understanding a product problem, structuring architecture, accelerated implementation, human code review, recovery and refinement, and shipping a reliable product. Use abstract diagrams, interface cards and code structures. The visual must emphasize human-led engineering and AI-assisted speed without depicting robots or artificial brains. No readable text.
