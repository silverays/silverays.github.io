# Zetmel — AI-Accelerated Software Engineering

The personal portfolio and professional website of [Safraz Razik](https://zetmel.com), founder of Zetmel — an AI-accelerated software engineering studio.

Built with [Astro](https://astro.build) and [Tailwind CSS](https://tailwindcss.com).

## Tech stack

- **Framework:** Astro 7
- **Styling:** Tailwind CSS 4 with OKLCH color tokens
- **Icons:** Lucide (UI) + Simple Icons (brand) via `astro-icon`
- **Content:** Astro Content Collections with Zod validation
- **Search:** Pagefind (post-build static search)
- **Dark mode:** 3-state (system/light/dark), no flash
- **Themes:** 12 switchable colour themes
- **Fonts:** Manrope (body), Outfit (display), JetBrains Mono (code)
- **SEO:** Structured data, Open Graph, sitemap, RSS

## Sections

| Route | Content |
|-------|---------|
| `/` | Homepage — hero, proof strip, capabilities, testimonials, selected work, blog, CTA |
| `/work/` | Case studies, projects and open-source work with filters |
| `/services/` | AI-accelerated engineering services |
| `/blog/` | Articles on engineering, AI, and development |
| `/about/` | Bio, approach, technology stack, interests |
| `/contact/` | Contact channels and direct email |
| `/resume/` | Full career timeline |

## Commands

| Command | Action |
|---------|--------|
| `npm run dev` | Start dev server at `localhost:4321` |
| `npm run build` | Build site + Pagefind search index |
| `npm run preview` | Preview production build |
