A feature-rich Vue.js file upload component that supports drag-and-drop, multi-file uploads, image previews, and customizable theming. Released as open-source to the Vue community.

## Architecture

The component was designed around a plugin-based architecture. The core handled file selection, validation, and state management while renderers handled the UI presentation. This separation meant users could customise the visual output without touching the file handling logic.

Key technical decisions:

- **Event-driven state machine** — file lifecycle modelled as explicit states
- **Slot-based theming** — Vue's slot system exposed every visual section
- **Progressive enhancement** — graceful fallback for older browsers

## What I Learned

Building an open-source component taught me more about API design than any client project. Every prop, event, and slot had to be justified because someone else would depend on it.
